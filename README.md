## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Git

### Local Setup
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd TaskKeeper
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
