*Define the high-level mission of the project. What problem does TaskKeeper solve, and for whom?*

## 🚫 Non-Goals
*Explicitly listing what we will **not** build helps maintain project focus and avoid scope creep.*

-   We will not support team collaboration features in version 1.0 (individual developer focus).
-   We will not build a native desktop app; it will remain a web app with PWA support.
-   We will not integrate with third-party sync services initially.

---

## 🗺️ Roadmap & Milestones

### 🏁 Milestone 1: MVP (Minimum Viable Product)
-   [ ] Keyboard-driven tasks creation and editing.
-   [ ] LocalStorage data persistence.
-   [ ] Markdown import/export of task lists.

### 🏁 Milestone 2: Cloud Sync & Search
-   [ ] User authentication (OAuth).
-   [ ] Real-time database sync.
-   [ ] Full-text search and filtering by tags.

### 🏁 Milestone 3: Advanced Integrations
-   [ ] Git commit message hook integration to close tasks via commit.
-   [ ] Custom theme support and keyboard layout customization.
